# Ubuntu中安装phpStorm


## 依赖
```
sudo apt-get default-jdk -y

```
linux版本的phpstorm是java开发的 依赖jdk环境

## 下载

```
wget -c http://download-cf.jetbrains.com/webide/PhpStorm-9.0.2.tar.gz
tar -zxvf PhpStorm-9.0.2.tar.gz
cd PhpStorm-9.0.2/bin
sh phpstorm.sh
```
