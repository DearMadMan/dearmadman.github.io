#caffeine in ubuntu

ubuntu系统常时间无操作就会进行休眠操作，这样我们看网页视频的时候会非常的烦恼，使用caffeine就可以取消这种休眠操作

##安装
```
sudo add-apt-repository ppa:caffeine-developers/ppa
sudo apt-get update
sudo apt-get install caffeine
```

##使用

在search里查找并启动caffeine , 桌面又上角会有一个咖啡的标志，右键激活就ok了
