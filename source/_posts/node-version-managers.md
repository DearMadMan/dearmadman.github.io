# node version managers

## n

```
sudo npm install -g n

```

`install`

```
sudo n stable

```

`install spacify version`

```
sudo n 0.12.7
```


`siwtch version`

```
n

```

`remove some version`

```
n rm 0.12.7
```

`install latest version`
```
n latest
```

`use specify version build js`
```
n use 0.12.7 app.js
```
